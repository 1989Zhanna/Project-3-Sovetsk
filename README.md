# Интерактивная карта г. Советск
https://1989Zhanna.github.io/Project-3-Sovetsk/
## Описание
Интерактивная карта по анализу недвижемости и землепользования города Советск Калининградской области

### Население:
- Всего: 38 270 чел.
- Дети 7-17 лет: 3 900-4 200 чел.
- Дети 0-6 лет: 2 100-2 500 чел.

### Жилой фонд:
- Всего: 1 028.41 тыс. кв.м.
- Износ >60%: 390.32 тыс. кв.м.
- Износ 40-60%: 286.34 тыс. кв.м.
- Аварийные дома: 23 шт. (6.9 тыс. кв.м.)

### Социальная инфраструктура:
- Школы: 7
- Детские сады: 10
- ОКН: 4

## Слои карты
- Граница города, данные из OSM
- Улично-дорожная сеть, данные из OSM
- Здания (полигоны), данные из OSM
- Землепользование, данные из OSM
- Зеленые зоны, данные из OSM
- Аварийные дома, сайт Администрации города Советск, выгрузка с Фонда развития территории, файл приложен

## Подложка
CartoDB Positron (светлая схема)

## Файлы
- `sovetsk_interactive_map.html` - основная карта
- `index.html` - страница для просмотра
- `README.md` - документация

## Как использовать
Откройте `index.html` в браузере
